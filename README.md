# Automação de Infraestrutura com AWS CloudFormation

Este projeto foi desenvolvido durante o laboratório da DIO com foco em automação de infraestrutura utilizando o serviço AWS CloudFormation.

📌 Objetivo

Automatizar a criação de recursos na AWS através de templates YAML, aplicando conceitos de Infraestrutura como Código (IaC).

⚙️ Recursos utilizados
Amazon EC2
Security Group
Key Pair
AWS CloudFormation

🧩 O que foi realizado
Criação de template automatizado
Provisionamento de infraestrutura via Stack
Configuração automática da EC2
Gerenciamento centralizado dos recursos
💡 Aprendizados

Durante o laboratório, consegui entender melhor como a automação reduz erros manuais e facilita a padronização de ambientes.

Também percebi a importância do versionamento da infraestrutura e da reutilização dos templates em diferentes cenários.

⚠️ Erros encontrados
Problemas de identação no YAML
AMI inválida na região escolhida
Falha de acesso SSH devido ao Security Group
